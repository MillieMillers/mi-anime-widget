# mi-anime-widget
Widget que actualiza los animes de temporada:
¿Cómo cambiar la temporada manualmente? Al inicio del <script> en index.html hay un bloque API_CONFIG con comentarios claros. Solo cambia mode: "specific" y ajusta year y season.
¿Cómo embeber en Notion? Usa el bloque /Embed de Notion y pega la URL de donde tengas hosteados los archivos (GitHub Pages, Vercel, etc.).
